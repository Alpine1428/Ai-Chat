# HolyWorld AI Assistant

Fabric 1.20.1 mod for HolyWorld moderators.

## Commands
- `/ai startlesson` - start learning
- `/ai stoplesson` - stop learning
- `/ai start` - enable auto-reply
- `/ai stop` - disable auto-reply
- `/ai status` - show status
- `/ai reset` - reset model

## Build
```
./gradlew build
```
